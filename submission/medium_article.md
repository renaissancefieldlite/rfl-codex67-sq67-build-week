## The Plain-English Finding

We found local Codex state and trace surfaces on one RFL Mac. The default lane was noisy and weak as an evidence surface. RFL built SQ67 as a clean receipt lane and measured better preservation with hashes, controls, recovery gates, and second-machine checks.

In normal words:

> Codex work can produce useful answers while leaving messy evidence trails. SQ67 turns the trail into a receipt book.

The narrow public claim is not that this proves motive or direct access to hidden model internals. The narrow claim is stronger:

> SQ67 proves a repeatable state-path testing harness and measurable evidence-preservation lift.

## What SQ67 Does

SQ67 is simple:

1. write the marker
2. hash the proof
3. recover it later
4. score whether it held
5. compare against controls

That turns a vague question like "did the state path hold?" into a reviewer-facing record.

## Results

## Why This Matters

AI agents are getting more capable, but the evidence trail around their behavior is still messy. If a system performs well but cannot preserve a clean receipt chain, reviewers cannot tell whether a result is stable, reproducible, accidental, or polluted by stale state.

RFL's approach is to make the state path inspectable without exposing private prompts or sensitive local records.
